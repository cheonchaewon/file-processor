# 프로세스 협력 파일 처리기

### 개요

- 클라이언트-서버 모델을 사용하여 파일의 각 줄 처리
- 클라이언트는 파일의 각 줄을 서버에 전송
- 서버는 지정된 처리 모드에 따라 각 줄을 처리한 후 결과를 클라이언트에 반환
- 처리 모드:
  - count: 문자 수 및 단어 수 계산
  - upper: 대문자로 변환
  - lower: 소문자로 변환
  - reverse: 문자열 뒤집기
- 사용법
- <input_file>: 처리할 텍스트 파일의 경로
- <mode>: count, upper, lower, reverse 중 하나

## 실행결과

### 클라이언트: count
### 클라이언트: upper
### 클라이언트: lower
### 클라이언트: reverse
